The first line consists of a single integer, denoting the number of integer strings.
Each line of the subsequent lines contains a real number.

Sort real number strings in descending order

## sample input
``` 
9
-100
50
0
56.6
90
0.12
.12
02.34
000.000
```


## sample output
``` 
90
56.6
50
02.34
0.12
.12
0
000.000
-100
```
