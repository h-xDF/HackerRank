In this problem you will be provided strings containing any combination of ASCII characters. You have to write a regular expression to find the valid IPs.

## sample input
```
000.12.12.034
121.234.12.12
23.45.12.56
00.12.123.123123.123
122.23
Hello.IP
```

## sample output
``` 
true
true
true
false
false
false
```
